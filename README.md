## プロジェクトの概要

オフィスの座席にネームプレートを置いて
誰がどこに座っているか明確になるアプリです。

操作方法は
まずユーザー登録ボタンを押して自分の名前を入力してボタンを押すと名前が表示されます
表示された名前を押してから座席を押すと名前が表示されます。
座席から名前を消す時は名前をクリックしてください。
これが基本的な使い方です。
